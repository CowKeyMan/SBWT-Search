Data class for the intervals. The chars_before_newline is a vector which gives the number of characters that a string contains, before the new one ends. newlines_before_newfile is a vector which tells us how many newlines we need before we need to start considering the next lines or sequences as originating from a new file. Note: the last character of both vectores will always be the max value (ULLONG_MAX), and both vectors are cumulative.
