File PositionsBatch.h
=====================

|exhale_lsh| :ref:`Parent directory <dir__home_runner_work_SBWT-Search_SBWT-Search_src_BatchObjects>` (``/home/runner/work/SBWT-Search/SBWT-Search/src/BatchObjects``)

.. |exhale_lsh| unicode:: U+021B0 .. UPWARDS ARROW WITH TIP LEFTWARDS



Has data created by the positions builder, which is a list of valid positions within a string sequence where the SBWT should search. A valid sequence is one where the string length is equal to the k-mer size. For example, given the string ABCDE and FGHIJ, and k-mer size 3, positions 0, 1, 2 are valid positions, but position 3, which is the k-mer starting at D, is not valid. Hence, our final position list will be: [0, 1, 2, 5, 6, 7].
