Parent template class for reading the list of integers provided by the indexing function. Provides a padded list of integers per seq and another list of indexes to indicate where each seq starts in our list of integers. Note: these classes expect the input to have the version number as the first item, and then the contents later. The format encoded in the file's header is read by another part of the code.
